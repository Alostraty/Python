# Задания цикла FOR
## Задание 1 
Рассматривается множество целых чисел, принадлежащих отрезку [1012; 9638], которые делятся на 7 и оканчиваются на 1. 
Найдите сумму таких чисел.
```
s = 0
for i in range(1012, 9639):
    if (i % 7 == 0) and (i % 10 == 1):
        s += i
print(s)
```
Программа должна вывести число 656943
## Задание 2
Рассматривается множество целых чисел, принадлежащих отрезку [1125; 7856], которые делятся а 3 и не делятся на 7 и 5. 
Найдите количество таких чисел.
```
count = 0
for i in range(1125, 7857):
    if (i % 3 == 0) and (i % 7 != 0) and (i % 5 != 0):
        count += 1
print(count)
```
Программа должна вывести число 1538.
## Задание 3
Рассматривается множество целых чисел, принадлежащих отрезку [100; 999], которые являются палиндромом (одинаково читаются как слева направо, так и справа налево). 
Найдите сумму таких чисел.
```
s = 0
for i in range(100, 1000):
    if str(i) == str(i)[::-1]:
        s += i
print(s)
```
Программа должна вывести число 49500.
## Задание 4
В программу вводится два натуральных числа: N и K. Выведите на экран последовательность, состоящую из N чисел с разностью K (последовательность начинается с единицы).
```
N = int(input())
K = int(input())
for i in range(N):
    print(1 + i * K, end=' ')
```
## Задание 5
В программу вводится одно натуральное число N. Выведите на экран все значения функции f(x) = x*x + 5x - 7 в целых точках на отрезке [0; N].
```
N = int(input())
for x in range(N + 1):
    print(x * x + 5 * x - 7, end=' ')
```
## Задание 6
В программу вводится натуральное число N, а затем N натуральных чисел. Требуется найти сумму чисел, которые делятся на 7 и являются трехзначными.
```
N = int(input())
total = 0
for _ in range(N):
    x = int(input())
    if 100 <= x <= 999 and x % 7 == 0:
        total += x
print(total)
```
В результате программа должна вывести одно число:сумму требуемых чисел.
## Задание 7
В программу вводится число N, после чего - N натуральных чисел, не превышающих 1000. 
```
N = int(input())
max_num = 0
for _ in range(N):
    x = int(input())
    if x % 7 == 0 and x % 9 != 0 and x > max_num:
        max_num = x
print(max_num)
```
Программа должна найти и вывести на экран максимальное число, кратное 7 и не кратное 9.
## Задание 8
В программу вводится число N, после чего - N натуральных чисел, не превышающих 1000.
```
N = int(input())
min_num = float('inf')
for _ in range(N):
    x = int(input())
    if 10 <= x <= 99 and oct(x)[-1] == '3' and x < min_num:

print(min_num)
```

Программа должна найти и вывести на экран минимальное двузначное число, восьмеричная запись которого оканчивается на 3.
## Задание 9
В программу вводится число N. Программа должна вывести два столбца чисел. В первом столбце – числа от 1 до N, во втором – числа от N до 1.
```
N = int(input())
for i in range(1, N + 1):
    print(i, N - i + 1)
```
## Задание 10
В программу вводится число N. Требуется определить, является ли это число простым.
```
N = int(input())
if N <= 1:
    print(False)
else:
    is_prime = True
    for i in range(2, int(N**0.5) + 1):
        if N % i == 0:
            is_prime = False
            break
    print(is_prime)
```
## Задание 11
В программу вводится натуральное число N. Требуется вывести на экран ряд чисел 1, 0.5, 0.25, 0.125,
0.0625, …, который содержит N элементов.
```
N = int(input())
num = 1.0
for i in range(N):
    if i == N - 1:
        print(num)
    else:
        print(num, end=' ')
    num /= 2
```
